My Journey to Building my first Portfolio Website

Published on 6 June 2026.

Today I deployed my first website using github and Vercel. Initially I had no idea about how to do it. But I went on with tutorials and did a lot of fixing the code again and again and finally my website is live. My website is very simple currently as I have not yet used any high level tech stack for making it. I used HTML and styled it with bootstrap. The website is currently very simple but I look forward to making it more complicated. In the long run, I see my website to be full of details in making it look very nice, colorful and highly interactive.
I will periodically update my website and once I learn Javascript, I will use my skills to make it more interactive. Now a days website is a fundamental need for any business. After the Corona period, every business is online and there is a great need for having an online presence for every business owners. The IT industry is changing and I look forward to being a part of it. And this step marks a great start in my journey. Now the only goal is to learn and keep updating my site.

Why did I decide to create a website

The fundamental reason that I made this website is that I wanted to have hands on experience with building a website from scratch and deploying it as in the tech it is very highly recommended to implement our skills to ensure that we have actually got the concepts.

What steps did I follow?

  1. First I designed my index.html file in my code editor.
  2. Then I made a github profile and updated my project to github.
    This step was challenging as I had to learn Git basics.
  3. Then I made a Vercel accounta and connected it with my Github.
  4. Finally I deployed my website at Vercel.
    Currently I am using free domain. But once I learn the tech , I will go for a paid domain.

It was challenging and at the same time very interesting. I really enjoyed doing it.
